# programu_uzturesana
Ieskaites darba trešā daļa priekšmetā Programmu uzturēšna - Bash scripta izveide un publicēšana Github.


Kā to izpildīt?
./jauns.sh dokumenta_nosaukums
Piemēram "./jauns.sh dokuments.txt"

Ko skripts dara? 
Skripts izveido jaunu skripta failu, kur jau tiek ievadīta pirmā rinda "#!/bin/bash"
Kā arī sākotnēji pārbauda vai lietotājs ir ievadījis faila nosaukumu, 
vai konkrētā mapē jau neeksistē tāds pats fails ar tādu pašu nosaukumu.
Piešķir izpildes tiesības visiem lietotājiem.

Pie veiksmīgas faila izveides izvada paziņojumu.
